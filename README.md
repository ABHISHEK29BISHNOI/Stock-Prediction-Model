
# Stock Price Prediction Web App

This project is a Machine Learning based web application that predicts the next day stock price using historical stock market data.

## Features

- Real-time stock data using yfinance
- Machine Learning model using Linear Regression
- Web interface using Flask
- Stock price visualization using Matplotlib

## Technologies Used

- Python
- Flask
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- HTML / CSS

## How to Run

```bash
pip install -r requirements.txt
python app.py
